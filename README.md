# Universal File Converter

A privacy-first, client-side file converter built with React, TypeScript, and Vite. All conversions happen directly in the browser — no files are uploaded to any server.

## Features

- Image conversion (JPEG, PNG, WebP, SVG)
- Document conversion (PDF, TXT, HTML, Markdown)
- Spreadsheet conversion (CSV, JSON, XML)
- Dark/light theme
- Drag-and-drop file upload
